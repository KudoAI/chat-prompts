# 💬 KudoAI / ai-personas

Epic prompts to turbo-charge your LLM chatbots.

## 💻 Usage

The easiest way is to import [chatgpt.js](https://chatgpt.js.org), then call:

```js
await chatgpt.actAs(persona)
```
